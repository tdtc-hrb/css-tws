Content and Border - 开关

content-box 是默认值

2025-04-11

Written by: tdtc

对 w and h 计算方式的选择:

  • content-box
只计算元素
  • border-box
包含border and padding

Content box

  • w
width = width of the content
  • h
height = height of the content

Border box

  • w
width = border + padding + width of the content
  • h
height = border + padding + height of the content
