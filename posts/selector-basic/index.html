子元素

当前元素的下一级元素。

element1 > element2 {
  css declarations;
}

后代元素

ol li

就表示当前元素（ol）下的所有li元素。

接续兄弟元素

element1 + element2 {
  css declarations;
}

后续兄弟元素

element1 ~ element2 {
  css declarations;
}
