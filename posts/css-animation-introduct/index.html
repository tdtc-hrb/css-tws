CSS Keyframes

  • Keyframe using percentages

Keyframe using percentages

        @keyframes scroll-left {
            0% {
                transform: translateX(100%);
            }
            100% {
                transform: translateX(-100%);
            }
        }

Keyframe using from-to keywords

@keyframes my-scroll {
  from {
    transform: translateX(0%);
  }

  to {
    transform: translateX(100%);
  }
}
